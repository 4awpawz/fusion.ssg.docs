Tokens

Tokens are declared in a template's front matter and their values replace all matching token markers that are found in the HTML document. Token marker syntax is the name of the token as declared in the template surrounded by single curly braces, as in {token name}.

Example: A template with two tokens. 

---
page: cat
tokens: {
    doctitle: Felix,
    breed: Persian
}
---
# This is content

This template declares two tokens, one named doctitle with a value of "Felix", and the other breed with a value of "Persian". All token markers in the HTML document that match {doctitle} and {breed} will be replaced with their respective token values.
