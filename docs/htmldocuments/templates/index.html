HTML DOCUMENTS

Templates

Required: Yes
Front Matter: Always
Type: .html, .md
Project Location: src/templates

Every HTML document requires a template and every template associates with a page using the page front matter property. If a template has its own content, its content will be inserted into the page that it associates with, replacing that page's template marker.

Example: A template associated with a page. 

---
page: main
---
# This is content

This template associates with the page found at src/pages/main.html.
