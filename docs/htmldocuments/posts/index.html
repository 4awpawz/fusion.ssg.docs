Posts

Required: No
Front Matter: Yes
Type: .html, .md
Project Location: src/templates/posts

Posts are specialized templates that can be useful for creating article and blog type HTML documents, or any type of HTML document that might benefit from the use of tags and categories.

Tags
Tags are useful for grouping similar posts to each other.
